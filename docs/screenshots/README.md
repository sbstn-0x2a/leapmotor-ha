# Installation Screenshot Checklist

Use these screenshots for the GitHub release page, HACS listing, and forum
posts. The committed screenshots are from a test Home Assistant instance and
must not contain VINs, real account email addresses, tokens, certificate
contents, precise home location, or other personal data.

Recommended files:

- `01-hacs-custom-repository.png`: HACS custom repository dialog with
  `https://github.com/kerniger/leapmotor-ha` and type `Integration`.
- `02-hacs-leapmotor-installed.png`: Leapmotor visible as installed in HACS.
- `03-add-integration.png`: Home Assistant add-integration search result for
  `Leapmotor`.
- `04-certificate-step.png`: certificate upload/paste step.
- `05-account-step.png`: account, Vehicle PIN, update interval, and optional
  ABRP fields.
- `06-device-overview.png`: created Leapmotor device with key entities.

Before replacing any screenshot later, check it for personal data first.
